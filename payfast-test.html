PayFast Sandbox Test

Signature generation is server-side only. Use the Next.js app to test:

  • Run the app and open /test-payfast or /test-payfast-simple
  • Those pages call POST /api/bookings/payfast-form, which builds the form and signs it with the canonical function in lib/payfast.ts

Do not use client-side signature generation. The only canonical implementation is generatePayFastSignature() in lib/payfast.ts (server-only).
